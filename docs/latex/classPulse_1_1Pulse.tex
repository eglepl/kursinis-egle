\subsubsection{Detailed Description}
Chemical reaction substance change of concentration at particular time. 

A \hyperlink{classPulse_1_1Pulse}{Pulse} tells that at time {\ttfamily time} we should set value of the compositor named {\ttfamily compositor\+\_\+name} to {\ttfamily value} in our simulation.
